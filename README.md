### Язык разметки MarkDown

**Параграф**  
автоматически __переносится__ на ~~следующую~~ строку
*Параграф 2*

_Параграф 3_

#### Нумерованный список

1. Первый пункт.
   1. Первый подпункт.
   2. Второй подпункт.
3. Второй пункт.

   Вложенный пункт
5. Третий пункт.

#### Маркерованный список

* Первый пункт.
   * Вложенный подпункт.
      - Вложенный во вложенный подпункт.
         - Вложенный во вложенный подпункт.
            - Вложенный во вложенный подпункт.
- Второй пункт.
+ Третий пункт.

```C#
   MyCkass myClass = new();
```

Мой строчный код `MyCkass myClass = new();` C# идёт в строке
